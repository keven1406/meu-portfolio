JavaScript – Estruturas de Repetição

Obs: Essa explicação é para pessoas que já estudaram lógica de programação. Está bem resumida. Se ainda não começou a estudar lógica, comece. A lógica facilita nosso aprendizado em níveis surreais!

Ao estudar programação, é muito importante saber como um programa é processado pelo computador. Sabemos que o inicio do algoritmo inicia-se na parte superior, logo quando digitamos a primeira linha de código. Quando o computador ler o nosso algoritmo, ele vai iniciar, logicamente, da primeira linha de código até a ultima. Porém, existe uma alternativa a isso, fazendo com que ele repita o código desejado. Como assim? Quando fazemos um programa que contenha uma caixa de texto, onde precisamos digitar a senha certa para acessar o conteúdo, estamos em uma estrutura de repetição. Quando erramos a palavra, o programa em vez de continuar, volta para o ponto onde estávamos, nos possibilitando reescrever a palavra.

Estrutura “Do while”

Vamos ver um exemplo bem simples:

const senha = 123; do { let inserir = Number(prompt("Digite a senha:")); } while (inserir != senha); alert("Bem vindo");

Aqui no exemplo, vemos que enquanto não digitamos o valor "123", o programa irá continuar pedindo a senha. Bem simples e legal né? Essa estrutura que vimos é a estrutura “do”. Ela funciona da seguinte maneira: Enquanto o resultado não for o esperado ela vai continuar o loop(repetindo) ate o resultado ser o correto.

do { } while (inserir != senha)

Ou seja, ele vai executar o bloco entre chaves ({ }) enquanto a variável “inserir” for diferente da variável “senha”. Tentem fazer isso com outros exemplos. É bem legal e da pra fazer uma infinidade de coisas.

While

Outra estrutura de repetição é a “While”. Vamos ver um código similar, mas dessa vez usando o while.

const senha = 123; while (senha != inserir) { var inserir = prompt("Digite a senha:"); } alert("Bem vindo");

Vai dar o mesmo resultado, a única diferença é que no método anterior, o programa executa o código para depois repeti-lo. Já com essa nova estrutura, o programa vai ver se precisa repetir e depois executa o conteúdo do seu bloco.

Vamos ver um exemplo ainda mais legal, baseado no código anterior:

const senha = 123; let contar = 5; while (cont > 0) { if (contar == 5) { let inserir = prompt(“Digite a senha:”); if ( inserir == senha ) { break ; } } else { let inserir = prompt("Você tem apenas: " + contar + " chances." + "\n Digite a senha:" ); if (inserir == senha) break ; } cont –; } if (inserir == senha) alert(“Bem vindo”); else if ( ! cont ) alert(“Suas chances acabaram”);

“For”

Vendo que a estrutura while é muito utilizadas nas linguagens de programação, o JavaScript trás uma estrutura resumida para facilitar nossa vida: A estrutura “for”. Vamos pegar o mesmo código anterior e colocar esse nova estrutura.

const senha = 123; let contar = 0; for (var contar = 5; cont > 0 ; cont - ) { if (contar == 5) { var inserir = prompt(“Digite a senha:”); if (inserir == senha) break ; } else { let inserir = prompt("Você tem apenas: " + contar + " chances." + "\n Digite a senha:" ); if (inserir == senha) break ; } } if (inserir == senha) alert(“Bem vindo”); else if ( ! contar ) alert("Suas chances acabaram");

Dessa forma podemos declarar a variável contadora, dizer a condição e usar o c–, que está monitorando a volta do loop. Tudo isso em uma linha de código só.
