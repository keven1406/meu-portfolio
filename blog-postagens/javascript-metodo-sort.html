JavaScript – Método de Array sort

Muitos se complicam quando tentam entender esse método, já que ele precisa de uma função. E é justamente nessa função que complica. Tudo é bem fácil, porém é questão de lógica.

Esse método permite ordenar os valores dos Arrays de forma crescente ou decrescente. Ele é muito fácil, já que só precisa acrescenta-lo a variável e pronto, estará ordenado de forma crescente. Vamos a um exemplo:

const numeros = [1,5,4,2,3,6]; const organizar = numeros.sort ( ); console.log (organizar);

Exibição no console: Array [1, 2, 3, 4, 5, 6]

Simples, não? Porem, quando usamos um numero que seja maior que nove ele não coloca na ordem correta. Vamos ver um exemplo com o mesmo código, sendo que agora um numero de dois dígitos:

const numeros = [1,5,4,2,3,6,35]; numeros.sort ( ); console.log (numeros);

Resultado no console: Array [ 1, 2, 3, 35, 4, 5, 6 ]

Notaram? Ele conta como se o 35 fosse igual ao 3, colocando ele próximo do 3. Para alterar isso precisamos usar uma função. Vamos ver um exemplo:

const ordenarArray = (numeroAnterior , numeroAtual) => { if (numeroAnterior > numeroAtual) { return 1; } if (numeroAnterior < numeroAtual) { return -1; } if (numeroAnterior === numeroAtual) { return 0; } } const numeros = [1,5,4,2,3,6,35]; console.log (numeros.sort (ordenarArray));

Resultado no console: Array [ 1, 2, 3, 4, 5, 6, 35 ]

Como podemos notar, o resultado no console agora está correto. Agora ele encara o 35 como um numero inteiro. Agora vamos fazer outra coisa divertida: Ordenar de forma decrescente. Ordenando de forma decrescente
