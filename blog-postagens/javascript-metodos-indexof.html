JavaScript – Métodos indexOf e lastIndexOf

Dois métodos bem interessantes e importantes para se programar em JavaScript são os metodos indexOf e LastIndexOf. Vamos aprender um pouco sobre eles.

O método IndexOf faz com que um valor estipulado por você seja buscado nos valores do Array. Resumindo, esse método faz uma busca de uma palavra que você deseja encontrar nos Arrays. Vamos ver um exemplo simples:

const nomes = [“Ana”,”João”,”Dora”,”Nina”,”João”]; const resposta = nomes.indexOf(“João”); document.write("Localização: " + resp + " . " + "Encontrado: " + nomes[resp]);

Resultado:

Localização: 2. Encontrado: João

É muito interessante como esse método consegue buscar o valor dentro do Array só digitando a palavra referente. Acha que isso é tudo? Está muito enganado. Vamos começar a busca a partir de um Array selecionado. No nosso caso vamos começar a busca a partir de “Dora” que é o Array nomes[2]. Para isso basta acrescentar (“João”, 2). Vamos ver um exemplo:

const nomes = ["Ana","João","Dora","Nina","João"]; const resposta = nomes.indexOf ("João",2); document.write ("Localização: " + resposta + ". Encontrado: " + nomes[resp]);

Resultado:

Localização: 4. Encontrado: João

O mesmo acontece com o método .lastIndexOf(). Porem, como o próprio nome já diz, ele começa a contagem do final do Array. Ex:

const nomes = [“Ana”,”João”,”Dora”,”Nina”,”João”]; const resposta = nomes.indexOf ("João",2); document.write ("Localização: " + resp + ". Encontrado: " + nomes[resp]);

Resultado:

Localização: 1. Encontrado: João

Caso não tenha entendido, a busca começou a partir do Array[2] que tem o valor "Dora", e começando a partir deste, seguiu em ordem decrescente até o joão, que está no Array nomes[1]. A diferença do indexOf para o lastIndexOf é justamente essa. Um começa a contagem do inicio e o outro do final.
