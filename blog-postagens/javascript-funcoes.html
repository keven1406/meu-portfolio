JavaScript básico – Funções

Funções são como mini programas dentro do seu programa principal que vão exercer um tipo de papel. Elas são variáveis que já vem pre-definidas ou que podem ser criadas por você ao decorrer do programa.

Exemplos de funções pre-definidas

console.log() – Mostra no console um log.

Math.min(a,b) – Mostra o menor valor dos dois números.

Math.max(a,b) – Mostra o maior valor dos dois números.

visNaN() – é NaN. Se acrescentar um operador lógico fica !éNaN (não é NaN ou não é numero).

valert() – Mostra uma janelinha de alerta.

confirm()– Mostra uma janela com a opção ok ou cancelar. Pode ser útil para treinos.

prompt() – Mostra uma opção com uma caixa de texto e a opção confirmar ou sair.

Number()– Transforma o valor que será inserido no type Number.

Boolean() – Transforma o valor que será inserido no type Booleano.

String() – Transforma o valor que será inserido no type String.

Esses são alguns exemplos de funções já existentes dentro do ambiente JavaScript.

Exemplo de código de cada função a cima
