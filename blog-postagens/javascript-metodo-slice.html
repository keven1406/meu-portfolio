JavaScript – Método slice

JavaScript possui alguns métodos bem interessantes que da para brincar bastante. Um dos Métodos de Array é o slice. Esse método permite que você “selecione” apenas valores de array escolhidas por você, para que assim, possa criar um novo array. Da para usar até pra esconder letras em uma String (cadeia de caracteres). Vamos ver esse código que utiliza o método slice.

const nome = “AnaMaria”; console.log (“Seu primeiro e segundo nome juntos são: “); console.log (nome); const primeiroNome = nome.slice (0, 3); const segundoNome = nome.slice (3, 8); console.log ("Seu nome separado é: " + " " + primeiroNome" + segundoNome + "." );

O resultado no console é:

Seu primeiro e segundo nome juntos são:
AnaMaria
Seu nome separado é: Ana Maria.

Observe que a constante primeiro nome está recebendo o valor que o metodo slice está retornando. Como passamos o primeiro argumento como 0, e o segundo sendo 3, ele irá retornar apenas os valores que estiverem na posição zero até a 3.

Interessante, né? Da para brincar bastante com as Arrays. São inúmeras as possibilidades. Então o que temos que fazer agora é: Praticar!!!!
