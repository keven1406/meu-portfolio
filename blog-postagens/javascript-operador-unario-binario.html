JavaScript básico – Operadores aritméticos Unários e Binários

Operadores estão e sempre esteve em nossas vidas. Desde nosso nascimento até a nossa morte. O universo em si é composto por valores e seus operadores. Podemos citar vários exemplos: Um nascimento, mais um dia de vida ou até mesmo um átomo compartilhando um elétron com outro átomo. Os operadores são o modo em que um elemento uni-se a outro para formar um novo elemento. Igual a um ingrediente que uni-se a outro para formar uma deliciosa comida. Nós vemos o uso de operadores explicitamente na matemática. Podemos notar que cada numero precisa unir-se a outro para gerar um resultado. Os operadores básicos são:

+ (adição), - (subtração), / (divisão) e * (multiplicação)

Você agora deve estar se perguntando: Por que esse cara me fez ler tudo isso só para explicar que operadores são as continhas que conhecemos na matemática? Calma, caro aprendiz. Apenas dei esse mar de comparações para que vocês entendam que o universo em si é parecido com a lógica de programação. Mas irei detalhar isso em outro post. Agora vamos ao que interessa.

No JavaScript existem dois tipos de operadores aritméticos: Os Unários e os Binários. O primeiro, refere-se a operações de apenas um valor. Enquanto o segundo refere-se, como o nome já diz, a dois valores. Como assim? Vamos entender melhor vendo um exemplo:

Binário: 1 + 1 = 2 /* Como podem notar, temos um operador “+” unindo dois números. Operadores binários sempre vão servir para calcular o valor de uma operação entre dois números. */

Unário: typeof(2) /*Aqui vemos um exemplo de operador Unário. Como podem ver, eles trabalham em cima de um único valor. */

Vamos ver como fica testando no console, lembrando que “//” significa o resultado mostrado no console.

Operador binário:

console.log (1 + 1);

Resultado:

//2

Operador Unário:

console.log (typeof(2));

//number

Obs: O operador “typeof()” diz se o valor é um número ou uma string.

Uma ideia legal, é transformar um operador Binário em Unário fazendo o seguinte processo:

console.log ( - ( 10 + 15));

//-25

Então, agora é com vocês. Vamos estudar e praticar esses operadores aritméticos.
