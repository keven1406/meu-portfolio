JavaScript – Arrays

Vimos que cada variável possui um valor podendo ser Booleano, Numérico ou String (cadeia de caracteres). Porem, ficamos muito limitados quando queremos inserir no nosso programa vários valores. Até onde aprendemos, para declarar uma variável precisamos digitar “var” mais o nome da variável, após isso colocar um sinal de igual e o valor da variável. Ex: const n = 1;

Mas e se agente precisar de várias variáveis com valores diferentes? Teríamos que declarar variável por variável? Não e é bem simples. Basta usar um Array. Um Array é como um grupo de variáveis reunidas em um único nome, bastando apenas para acessa-lo digitar o nome do Array mais o número do local onde o valor que queremos está. Vamos entender melhor com esse exemplo:

const numeros = [1,2,3,4];

Para acessar o valor “1”, precisamos por em um console.log, por exemplo: numeros[0]. Por que precisamos chamar o Array [0] para acessar o valor 1? Por que o endereço dos valores começam por zero. O valor um está na posição “0” do Array, assim como o valor dois está na posição 1 do array e assim sucessivamente.

Existem várias formas de usar um array. Vamos ver uma bem simples.

const numeros = [1,2,3]; for (let atual = 0; atual < 2; atual++) { console.log (numeros[atual]); }

O código a cima mostra no console o valor que cada Array tem. O valor do Array[0] é 1, o valor do Array[1] é 2 e o valor do Array[2] é 3.
